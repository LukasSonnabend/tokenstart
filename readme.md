# TokenStart Alpha

Main development version of TokenStart

## Installation

Use the package manager [npm] in folder /tokenstartauth & /tokenstartfrontend

```bash
npm install 
```

## Usage
In folder /tokenstartauth

```javascript
npm run devStart

```

In folder /tokenstartfrontend

```javascript
npm run start

```
